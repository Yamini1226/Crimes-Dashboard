🚨 Crime Analysis Dashboard (2001–2010) | Power BI | Data Analytics Project
📌 Project Overview

This project is an interactive Crime Analysis Dashboard built using Power BI to analyze crime data across India from 2001 to 2010.

The dashboard transforms raw NCRB crime data into meaningful insights by identifying long-term trends, high-crime districts, place-wise crime distribution, juvenile involvement, and victim statistics.

The goal of this project was to apply data cleaning, transformation, and visualization techniques to derive actionable insights from multi-table crime datasets.

🎯 Business / Analytical Objectives

Analyze overall crime trends over a 10-year period

Identify high-crime districts and geographical hotspots

Examine crime occurrence by place type

Study juvenile apprehension patterns

Track reported victims trends over time

Enable dynamic filtering for interactive exploration

🛠️ Tech Stack

Power BI – Dashboard design & visualization

DAX – KPI calculations & analytical measures

Microsoft Excel – Data preprocessing & structuring

CSV Files – Multi-table dataset integration

📂 Data Source

Dataset: Crime in India

Source: Kaggle

Link: https://www.kaggle.com/datasets/rajanand/crime-in-india

The dataset includes district-wise crimes, juvenile apprehensions, crimes by place of occurrence, and victims statistics collected by the National Crime Records Bureau (NCRB), India.

📊 Key Features

✔️ Overall Crime Trend Analysis (2001–2010)
✔️ District-wise Crime Distribution with Geographic Mapping
✔️ Crimes by Place Type (Overall Share & Trend)
✔️ Juvenile Apprehension Analysis
✔️ Victims Trend Visualization
✔️ KPI Cards for High-Level Summary
✔️ Interactive Slicers (Year, State/District)
✔️ Clean, Story-Driven Dashboard Layout

📈 Key Insights Derived

Identified year-wise fluctuations and long-term crime growth patterns

Highlighted top contributing districts to overall crime volume

Observed place-wise crime dominance trends

Tracked changes in juvenile involvement across years

Detected variations in reported victim counts over time
